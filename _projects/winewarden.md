### Overview
Comprehensive security sandbox for Windows games on Linux with real-time monitoring, filesystem virtualization, and dynamic trust scoring.

### Highlights
- Real-time filesystem virtualization
- Network monitoring and DNS awareness
- Process sandboxing and policy engine
- Dynamic trust scoring (0-100)
- Interactive TUI dashboard
- Landlock LSM and Seccomp integration
- Built by S1BGr0up organization
